# Исправление стартового баланса

## Изменение
Новые пользователи теперь создаются с **0 NDN** и **100 Gas** для игры Miner

## Что изменено

### 1. NDN Баланс = 0
**Файлы:**
- `server-deployment/routes/users.js`
- `server-deployment/routes/auth.js`

**Было:**
```javascript
balance_ndn: 100,
```

**Стало:**
```javascript
balance_ndn: 0,
```

### 2. Miner Gas = 100
**Файл:** `server-deployment/routes/miner.js`

**Настройка:**
```javascript
const initialMinerData = {
  ndnGas: 100, // Стартовый Gas для покупки первой фермы
  energy: 100,
  ...
};
```

## Как это работает

### Старые пользователи
- Сохраняют свой текущий баланс NDN
- Изменения не затрагивают их

### Новые пользователи
- Создаются с **0 NDN**
- В игре Miner получают **100 Gas** для старта
- Могут купить 2 CPU Miner (50 Gas каждый)

## Логика игры

### Баланс NDN
- Используется для покупок в магазине
- Начисляется за рефералов
- Начисляется за ежедневные награды
- Можно купить за Telegram Stars

### Баланс Gas (Miner)
- Используется для покупки ферм
- Зарабатывается майнингом
- Первая ферма стоит 50 Gas
- Стартовый баланс: 100 Gas

## Стартовые ресурсы

### До изменения
```javascript
balance_ndn: 100  // ❌ Слишком много
ndnGas: 100       // ✅ Для игры
```

### После изменения
```javascript
balance_ndn: 0     // ✅ Начинаем с нуля
ndnGas: 100       // ✅ Для покупки ферм
```

## Преимущества

1. **Более честная экономика** - пользователи должны заработать NDN
2. **Быстрый старт в Miner** - можно сразу начать майнить
3. **Мотивация** - нужно играть чтобы получить NDN

## Проверка

Попросите нового пользователя:

1. Открыть Mini App
2. Проверить баланс NDN = 0
3. Зайти в игру Miner
4. Проверить баланс Gas = 100
5. Купить CPU Miner за 50 Gas
6. Увидеть баланс Gas = 50

## Важно

- Старые пользователи сохраняют свой баланс
- Новые пользователи получают 0 NDN
- В игре Miner все получают 100 Gas
- Первая ферма стоит 50 Gas (2 фермы на старт)
